Branches In Git

Branch is used for if you want to modified the file but also want to save the original file. For this branch is used. You make a branch and do the change their without changes the original content.

  • Git Branch(used to see the branches.By default their is branch name "master")
  • Git branch branch_name(used to make new branch)

  • Git Checkout branch_name(used to switch branch)

If you want to merge the code of one branch to another use the following command.

  • Git merge branch_name
  • Git branch -D branch_name(used to delete the branch)
